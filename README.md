# watermark

## 1. Install libraries

```
pip install -r requirements.txt
```

***

## 2. Run app.py

```
python3 app.py
```

- Head to `127.0.0.1/docs`

## 3. Try the app

- Click ``try it`` button
- You can change some parameters but we recommend between 1 and 10 (except **color**)
- Upload image and click ```Execute```
- The **water mark image** will appear below.
